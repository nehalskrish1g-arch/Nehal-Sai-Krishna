Pseudocode

algorithm Prim(Graph, startNode):
for each u in V: key[u] = ∞, parent[u] = null
key[startNode] = 0
MST = {}
while MST is not full:
u = node with min key not in MST
add u to MST
for each neighbor v of u:
if v not in MST and weight(u,v) < key[v]:
key[v] = weight(u,v)
parent[v] = u

* Nodes sorted by ID here, but algorithm picks based on Min Key.

Complexity Analysis

Time (Adj Matrix) O(V²)
Space O(V)

This implementation uses an Adjacency Matrix approach. Using a Binary Heap (Priority Queue) and Adjacency List would yield O(E log V).
